# Olist E-Commerce Analysis 🛒

Data analysis on Brazilian e-commerce public dataset 

## Business Questions
1. When do customers order the most? (hour & day pattern)
2. Does delivery speed affect customer satisfaction?
3. What payment methods do customers prefer?

## Key Insights
1. **Peak order time**: Monday to Wednesday, 9AM-6PM
2. **Delivery speed = satisfaction**: Orders delivered 13 days 
   earlier than estimated get 5-star ratings vs only 4 days 
   earlier for 1-star ratings - This means more early more rating.
3. **Credit card dominates**: 76% of transactions use credit card, 
   and has the highest average transaction value

## Dataset
[Olist Brazilian E-Commerce Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
